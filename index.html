Instrucciones para Repositorio Compartido en GitHub

1. Configuración Inicial

  1. Crear un repositorio local: git init
  2. Añadir los archivos y carpetas indicados: git add .
  3. Realizar el primer commit: git commit -m "Primer commit: configuración inicial"

2. Repositorio Remoto en GitHub

  1. Crear un repositorio remoto en GitHub:
  2. Vincular el repositorio local con el remoto: git remote add origin https://github.com/usuario/nombre-del-repositorio.git
  3. Subir el proyecto al repositorio remoto: git push -u origin main

3. Trabajo Colaborativo

  1. Cada miembro del grupo debe:
    • Crearse como colaborador del repositorio:

      Para agregar a alguien como colaborador, debes acceder al repositorio en GitHub, ir a la sección "Settings" (Configuración), luego "Manage access" y añadir colaboradores por su nombre de usuario o correo electrónico.

    • Crear una rama con su nombre: git checkout -b nombre-de-tu-rama
    • Trabajar exclusivamente en su propia rama:
    • Documentar cada cambio realizado mediante commits descriptivos: git commit -m "Descripción clara del cambio realizado"
